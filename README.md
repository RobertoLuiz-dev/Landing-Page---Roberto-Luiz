# Portfolio Roberto Luiz

Landing page estatica para centralizar apresentacao profissional, links de projetos e contato.

## Rodar localmente

```bash
npm install
npm run dev
```

## Estrutura

- `index.html`: pagina principal
- `style.css`: identidade visual e responsividade
- `assets/`: espaco para imagens e icones futuros
